# 第8章 if文

## 学習目標

- if文の基本を理解する
- 条件によって処理を分けられる
- `else` を使って条件に合わない場合の処理を書ける
- `else if` を使って複数の条件を扱える
- 比較演算子・論理演算子を条件式で使える

---

## if文とは

`if` 文は、条件によって実行する処理を変えるための文です。

たとえば、年齢が18歳以上なら「成人です」と表示する、  
点数が60点以上なら「合格です」と表示する、というような処理ができます。

---

## if文の基本形

if文は次のように書きます。

```java
if (条件式) {
    条件がtrueのときに実行する処理
}
```

条件式の結果が `true` のときだけ、 `{}` の中の処理が実行されます。

---

## if文の例

```java
public class Main {
    public static void main(String[] args) {
        int age = 20;

        if (age >= 18) {
            System.out.println("成人です");
        }
    }
}
```

実行結果：

```text
成人です
```

`age >= 18` は `true` なので、  
`System.out.println("成人です");` が実行されます。

---

## 条件式とは

条件式とは、結果が `true` または `false` になる式のことです。

```java
age >= 18
```

この式は、`age` が18以上なら `true`、  
18未満なら `false` になります。

---

## 比較演算子を使う

if文では、比較演算子をよく使います。

| 演算子 | 意味 | 例 |
|---|---|---|
| `==` | 等しい | `score == 100` |
| `!=` | 等しくない | `score != 0` |
| `>` | より大きい | `score > 80` |
| `<` | より小さい | `score < 60` |
| `>=` | 以上 | `age >= 18` |
| `<=` | 以下 | `age <= 12` |

---

## else文

`else` を使うと、条件が `false` のときの処理を書けます。

```java
if (条件式) {
    条件がtrueのときの処理
} else {
    条件がfalseのときの処理
}
```

---

## else文の例

```java
public class Main {
    public static void main(String[] args) {
        int age = 15;

        if (age >= 18) {
            System.out.println("成人です");
        } else {
            System.out.println("未成年です");
        }
    }
}
```

実行結果：

```text
未成年です
```

`age >= 18` は `false` なので、  
`else` の中の処理が実行されます。

---

## 入力された値で判定する

`Scanner` を使うと、入力された値によって処理を分けられます。

```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("年齢を入力してください: ");
        int age = scanner.nextInt();

        if (age >= 18) {
            System.out.println("成人です");
        } else {
            System.out.println("未成年です");
        }

        scanner.close();
    }
}
```

実行例：

```text
年齢を入力してください: 20
成人です
```

---

## else if文

条件を複数に分けたい場合は、`else if` を使います。

```java
if (条件式1) {
    条件式1がtrueのときの処理
} else if (条件式2) {
    条件式2がtrueのときの処理
} else {
    どの条件にも当てはまらないときの処理
}
```

---

## else if文の例

点数によって評価を分けてみます。

```java
public class Main {
    public static void main(String[] args) {
        int score = 85;

        if (score >= 90) {
            System.out.println("評価: A");
        } else if (score >= 80) {
            System.out.println("評価: B");
        } else if (score >= 70) {
            System.out.println("評価: C");
        } else if (score >= 60) {
            System.out.println("評価: D");
        } else {
            System.out.println("評価: F");
        }
    }
}
```

実行結果：

```text
評価: B
```

`score` は `85` なので、`score >= 80` の条件に当てはまります。

---

## 条件は上から順番に判定される

`else if` は、上から順番に条件を確認します。

一度条件に当てはまると、そのあとの条件は確認されません。

```java
int score = 95;

if (score >= 60) {
    System.out.println("合格");
} else if (score >= 90) {
    System.out.println("高得点");
}
```

この場合、`score >= 60` が先に `true` になるため、  
`score >= 90` は確認されません。

実行結果：

```text
合格
```

高得点も判定したい場合は、条件の順番を変える必要があります。

```java
int score = 95;

if (score >= 90) {
    System.out.println("高得点");
} else if (score >= 60) {
    System.out.println("合格");
}
```

実行結果：

```text
高得点
```

---

## 条件の順番に注意する

条件を書くときは、より限定的な条件を先に書くことが多いです。

良い例：

```java
if (score >= 90) {
    System.out.println("A");
} else if (score >= 80) {
    System.out.println("B");
} else if (score >= 70) {
    System.out.println("C");
}
```

このように、大きい条件から順番に書くと自然です。

---

## 論理演算子を使う

複数の条件を組み合わせるときは、論理演算子を使います。

| 演算子 | 意味 | 例 |
|---|---|---|
| `&&` | かつ | `age >= 18 && age < 65` |
| `||` | または | `age < 18 || age >= 65` |
| `!` | ではない | `!isStudent` |

---

## `&&` の例

`&&` は、両方の条件が `true` のときだけ `true` になります。

```java
public class Main {
    public static void main(String[] args) {
        int age = 20;
        boolean hasTicket = true;

        if (age >= 18 && hasTicket) {
            System.out.println("入場できます");
        } else {
            System.out.println("入場できません");
        }
    }
}
```

実行結果：

```text
入場できます
```

---

## `||` の例

`||` は、どちらか一方でも `true` なら `true` になります。

```java
public class Main {
    public static void main(String[] args) {
        int age = 12;

        if (age < 13 || age >= 65) {
            System.out.println("割引対象です");
        } else {
            System.out.println("通常料金です");
        }
    }
}
```

実行結果：

```text
割引対象です
```

---

## `!` の例

`!` は、`true` と `false` を反対にします。

```java
public class Main {
    public static void main(String[] args) {
        boolean isLoggedIn = false;

        if (!isLoggedIn) {
            System.out.println("ログインしてください");
        }
    }
}
```

実行結果：

```text
ログインしてください
```

`isLoggedIn` は `false` ですが、  
`!isLoggedIn` によって `true` として判定されます。

---

## 文字列の比較に注意

Javaで文字列を比較するときは、`==` ではなく `equals()` を使います。

```java
String password = "java";

if (password.equals("java")) {
    System.out.println("ログイン成功");
}
```

`==` は、文字列の内容比較では思った通りに動かないことがあります。

```java
String password = "java";

if (password == "java") {
    System.out.println("ログイン成功");
}
```

この書き方は避けましょう。

---

## 入力された文字列を比較する

```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("パスワードを入力してください: ");
        String password = scanner.next();

        if (password.equals("java")) {
            System.out.println("ログイン成功");
        } else {
            System.out.println("ログイン失敗");
        }

        scanner.close();
    }
}
```

実行例：

```text
パスワードを入力してください: java
ログイン成功
```

## if文の中にif文を書く

if文の中に、さらにif文を書くこともできます。

```java
public class Main {
    public static void main(String[] args) {
        int age = 20;
        boolean hasTicket = true;

        if (age >= 18) {
            if (hasTicket) {
                System.out.println("入場できます");
            } else {
                System.out.println("チケットが必要です");
            }
        } else {
            System.out.println("18歳未満は入場できません");
        }
    }
}
```

このような書き方を **ネスト** といいます。

ただし、ネストが深くなりすぎると読みにくくなるため注意しましょう。

---

## よくあるエラー

| エラー | 原因 | 解決方法 |
|---|---|---|
| `';' expected` | セミコロン忘れ | 文の最後を確認する |
| `else without if` | `else` に対応する `if` がない | `{}` の位置を確認する |
| `bad operand types` | 条件式の型が正しくない | `true` / `false` になる式を書く |
| `cannot find symbol` | 変数名のミス | 宣言した変数名を確認する |
| 文字列比較がうまくいかない | `==` を使っている | `equals()` を使う |

---

## まとめ

- if文は条件によって処理を分けるために使う
- 条件式の結果は `true` または `false` になる
- `else` は条件に合わない場合の処理を書く
- `else if` は複数の条件を判定するときに使う
- 条件は上から順番に判定される
- 複数の条件は `&&` や `||` で組み合わせられる
- 文字列を比較するときは `equals()` を使う
