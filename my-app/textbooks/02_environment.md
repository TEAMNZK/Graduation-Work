# 第2章 開発環境を準備する

## 学習目標

- Javaの開発環境を準備できる
- Javaプログラムを実行できる状態にする
- エディタの基本操作を理解する

---

## 開発環境とは

Javaでプログラムを作成するには、
次のようなソフトウェアが必要です。

| ソフトウェア | 役割 |
|---|---|
| JDK | Javaを実行・コンパイルする |
| エディタ | プログラムを書く |
| ターミナル | コマンドを実行する |

---

## インストールを確認する

ターミナル（コマンドプロンプト）を開き、
次のコマンドを実行します。

```bash
java -version
```

以下の結果が表示された場合、JDKというツールをインストールする必要があります。
```
'java' は、内部コマンドまたは外部コマンド、
操作可能なプログラムまたはバッチ ファイルとして認識されていません。
```
以下のような結果が表示された場合、すでにJDKがインストールされています。
```
openjdk version "25" 2025-09-16
OpenJDK Runtime Environment (build 25+36-3489)
OpenJDK 64-Bit Server VM (build 25+36-3489, mixed mode, sharing)
```
> 注意:
> 使用しているJDKの種類やバージョンによって表示される内容は異なる場合があります。
---

## JDKをインストールする

### JDKとは

JDK（Java Development Kit）は、
Javaプログラムを開発するためのツールです。

Javaを書くためには必須です。

---

### JDKをダウンロードする

以下の公式サイトへアクセスします。

- Oracle JDK
- OpenJDK

自分のOSに合ったインストーラーをダウンロードします。
